 77 <span id="L22" class="LineNr"> 22 </span><span class="Normal">===</span><span class="Title"> Attributes that can be assigned to task </span><span class="Normal">===</span>
 78 <span id="L23" class="LineNr"> 23 </span><span class="Identifier">  - </span>Tags
 79 <span id="L24" class="LineNr"> 24 </span><span class="Identifier">    - </span>Tags are arbitrary words associated with a task. Use + to add a tag and a - to remove a tag
 80 <span id="L25" class="LineNr"> 25 </span>      from a task:

 83 <span id="L28" class="LineNr"> 28 </span><span class="Identifier">    - </span>A task can have any number of tags.
 84 <span id="L29" class="LineNr"> 29 </span>
 85 <span id="L30" class="LineNr"> 30 </span><span class="Identifier">  - </span>Project
 86 <span id="L31" class="LineNr"> 31 </span><span class="Identifier">    - </span>Each task can be associated with at most one project.
 87 <span id="L32" class="LineNr"> 32 </span><span class="Identifier">      * </span>task add project:Home Water the flowers
 88 <span id="L33" class="LineNr"> 33 </span><span class="Identifier">    - </span>Project is just a plain text attribute.
 89 <span id="L34" class="LineNr"> 34 </span><span class="Identifier">    - </span>Subprojects can be specified by a . syntax.
 90 <span id="L35" class="LineNr"> 35 </span><span class="Identifier">      * </span>task add project:Home.Kitchen Clean the sink
 91 <span id="L36" class="LineNr"> 36 </span>
 92 <span id="L37" class="LineNr"> 37 </span><span class="Identifier">  - </span>Priority
 93 <span id="L38" class="LineNr"> 38 </span><span class="Identifier">    - </span>Priority attribute specifies the priority for the task.
 94 <span id="L39" class="LineNr"> 39 </span><span class="Identifier">      * </span>task add pri:H Water the flowers
 95 <span id="L40" class="LineNr"> 40 </span><span class="Identifier">    - </span>By default, only values H, M or L are supported (plus empty).
 96 <span id="L41" class="LineNr"> 41 </span>
 97 <span id="L42" class="LineNr"> 42 </span><span class="Identifier">  - </span>Annotations
 98 <span id="L43" class="LineNr"> 43 </span><span class="Identifier">    - </span>An annotation is a note added to an existing task, and is in the form of text string.
 99 <span id="L44" class="LineNr"> 44 </span><span class="Identifier">      * </span>task add Buy the milk

103 <span id="L48" class="LineNr"> 48 </span><span class="Identifier">  - </span>Dates (due date, until date, scheduled date)
104 <span id="L49" class="LineNr"> 49 </span><span class="Identifier">    - </span>Different kinds of dates are available:
105 <span id="L50" class="LineNr"> 50 </span><span class="Identifier">      - </span>Use &quot;due date&quot; to specify the exact date by which a task must be completed.
106 <span id="L51" class="LineNr"> 51 </span><span class="Identifier">      - </span>The &quot;scheduled date&quot; represents the earliest opportunity to work on a task.
107 <span id="L52" class="LineNr"> 52 </span><span class="Identifier">      - </span>The &quot;wait date&quot; represents the earliest opportunity the task should show up on the
108 <span id="L53" class="LineNr"> 53 </span>        task list.
109 <span id="L54" class="LineNr"> 54 </span><span class="Identifier">      - </span>The &quot;until date&quot; specifies a date after which the tasks self-destructs.

167 <span id="L112" class="LineNr">112 </span><span class="Normal">===</span><span class="Title"> Task Identifiers </span><span class="Normal">===</span>
168 <span id="L113" class="LineNr">113 </span>Two ways: using ID or UUID
169 <span id="L114" class="LineNr">114 </span><span class="Identifier">- </span>ID is the line number in the pending data file, the UUID is permanent.
170 <span id="L115" class="LineNr">115 </span><span class="Identifier">- </span>UUID (Universally Unique IDentifier) is a 128=bit number used to Uniquely Identify a task.
171 <span id="L116" class="LineNr">116 </span>
172 <span id="L117" class="LineNr">117 </span>A UUID never changes, and is not modifiable.
173 <span id="L118" class="LineNr">118 </span>

185 <span id="L130" class="LineNr">130 </span><span class="Normal">===</span><span class="Title"> Urgency </span><span class="Normal">===</span>

187 <span id="L132" class="LineNr">132 </span>

189 <span id="L134" class="LineNr">134 </span>
190 <span id="L135" class="LineNr">135 </span>Taskwarrior uses a weighted linear expression to calculate urgency, and you can tweak the
191 <span id="L136" class="LineNr">136 </span>coefficient.

193 <span id="L138" class="LineNr">138 </span><span class="Identifier">  * </span>urgency.active.coefficient         4.0
194 <span id="L139" class="LineNr">139 </span><span class="Identifier">  * </span>urgency.age.coefficient            2.0
195 <span id="L140" class="LineNr">140 </span><span class="Identifier">  * </span>urgency.age.max                    365
196 <span id="L141" class="LineNr">141 </span><span class="Identifier">  * </span>urgency.annotations.coefficient    1.0
197 <span id="L142" class="LineNr">142 </span><span class="Identifier">  * </span>urgency.blocked.coefficient        -5.0
198 <span id="L143" class="LineNr">143 </span><span class="Identifier">  * </span>urgency.blocking.coefficient       8.0
